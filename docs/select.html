Select columns

Two ways to create DataFrame with a subset of columns:

indexing:

df[df.age, df.weight]
val age by column<Int>() val weight by column<Int?>() df[age, weight]
df["age", "weight"]

See DataFrame indexing

selecting:

df.select { age and weight }
val age by column<Int>() val weight by column<Int?>() df.select { age and weight } df.select(age, weight)
df.select { "age" and "weight" } df.select("age", "weight")

See column selectors
